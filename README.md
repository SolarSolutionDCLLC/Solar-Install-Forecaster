Solar Install Forecaster ☀️

📌 Overview

The Solar Install Forecaster is a robust, single-page web application designed to help solar installation teams track, manage, and forecast their deployment capacity. Built with a modern, responsive UI, it provides real-time insights into crew allocations, commercial versus residential backlogs, weather-related delays, and installation costs.

✨ Features

KPI Dashboard: High-level visual key performance indicators (KPIs) for active goals, close rates, and total kW installed.
Crew & Timeline Management: Dynamically configure internal and external crews, manage capacity shares, and block out custom timeline entries for specific dates.
Backlog Tracking: Dedicated modules to monitor residential pipelines and commercial (Solar Green) backlogs alongside future allocations.
Interactive Visualizations: Integrated charting to map out forecasted growth, capacity limits, and installation pacing over time.
Weather Impact Adjustments: Configurable sliders to calculate weather-related efficiency losses and adjust installation forecasts dynamically.
Cost Calculation Engine: Built-in, auto-tallying tables to break down job costs, subtotals, and bottom-line estimates.
State Management: Includes an auto-save feature to preserve your work locally, plus options to manually Import/Export your forecast data as a file.
🛠️ Technologies Used

HTML5: Semantic structure.
CSS3: Built entirely with custom CSS (no external CSS frameworks). Utilizes CSS Variables (:root), Flexbox, and CSS Grid for a highly responsive "Sidebar + Main Content" layout.
Vanilla JavaScript: Handles state logic, math calculations, DOM manipulation, and local browser storage.
Chart.js (v4.4.1): Embedded via CDN for rendering the interactive data charts.
Google Fonts: Uses DM Sans and DM Mono for clean, readable typography.
🚀 Getting Started

This application is completely client-side and requires no build tools, package managers, or server environments.
